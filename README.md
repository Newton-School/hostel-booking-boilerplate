## Hostel Allotment
This is a hostel allotment app. Complete the components as directed.

Sign Up - New students should register themshelves here using Email and Password.<br>
Login - The registered users should be able to login using Email and Password.<br>
Welcome - If the user already has a room alloted , the welcome page should pop up.<br>
Card - The user should choose Boy or Girl for hostel allotment.<br>
Bhostel - Select one of the six Boy's hostel.<br>
Ghostel - Select one of the six Girl's hostel.<br>
Room -  Select the floor and room here.<br>
Success - Once the room is alloted the success page should appear.<br>
